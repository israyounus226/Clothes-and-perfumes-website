# Clothing and Perfumes Project

This is a pair project developed for:
- Programming Fundamentals (Python)
- AICT (HTML & CSS)

## Project Description
The project consists of two separate parts:
1. A console-based Python program implementing basic programming concepts.
2. A frontend website for a clothing and perfumes system developed using HTML and CSS.

## Technologies Used
- Python
- HTML
- CSS

## Note
The frontend and Python modules are developed separately and are not linked.
